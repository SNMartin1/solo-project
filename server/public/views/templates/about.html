About Game Vault

A Board Game App where board game lovers can keep track their library of games and their gaming sessions! The app can track how many times you have played each board game and keep track of how many times you have won.


Technologies Used

  • MongoDB
  • Express
  • AngularJS
  • Node.js
  • Bootstrap
  • Passport Authentication
  • Filestack API
  • Heroku
  • mLabs

Stretch Goals

  • Bring in boardgamegeek API for box cover art, publishing information, and information about rules.
  • Add ability to track if your game has been loaned out to a friend.
  • Add ability to link with other user's accounts (become friends, be able to tag them in sessions played.)
  • Ability for the user to upload pictures to each session.
  • Connect with Chase's app for board game meetups.
